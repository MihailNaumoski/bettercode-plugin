# BetterCode

MCP skill server with **60+ curated skills** and **20+ agents** for Claude Code.

## Installation

```bash
# Add marketplace
/plugin marketplace add bettercode-studio/bettercode-marketplace

# Install
/plugin install bettercode@plugin
```

## Commands

| Command | Description |
|---------|-------------|
| `/skills` | Browse skills by category |
| `/review` | Code review |
| `/security-review` | OWASP security audit |
| `/test <file>` | Generate tests |
