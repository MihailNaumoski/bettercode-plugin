Use the `mcp__bc-skills__bc_menu` tool to show the main BetterCode skills menu.

Then help the user explore available skills by category or search for specific skills using the `search` tool.

Available categories include:
- Frontend (React, responsive design, animations, Figma-to-code)
- Backend (API design, webhooks, performance optimization)
- Database (modeling, optimization, migrations)
- Security (auth patterns, OWASP review, hardening)
- DevOps (CI/CD, Docker, Kubernetes)
- Testing (unit tests, E2E, mocking)
- Architecture (ADRs, patterns, tech stack decisions)

Guide the user to find the right skill for their task.
